# debuxOS

Minimal x86 Multiboot kernel тАФ VGA GUI, PS/2 CLI, RAM VFS, mock network.

## Build

- [ ] Install tools: `nasm`, `i686-elf-gcc`, `i686-elf-ld`, `qemu-system-i386`
- [ ] `make` тАФ assembles + compiles + links тЖТ `debuxos.bin`
- [ ] `make run` тАФ boots the binary in QEMU; type `help`, `ls`, or `ping`
